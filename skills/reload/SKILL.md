# reload

Reload team, jinxes, or NPCs from disk. Usage: /reload (all), /reload team, /reload jinx <name>

## Inputs

- `type` (default: `''`)
- `name` (default: `''`)

## Usage

```
/run_jinx jinx_ref=reload input_values={"type": "", "name": ""}
```
